▶ Methodology

Unit converter scope

  • This panel provides quick bedside unit conversions for perfusion-related flow and pressure values.
  • It is a calculation aid only and does not replace patient-specific interpretation or institutional protocols.
  • All computations are performed locally in the browser from the entered value and selected unit.

Flow methodology (base input)

  • Base input uses flow in L/min, with optional weight in kg.
  • mL/min = L/min × 1000
  • mL/kg/min = (L/min × 1000) / weight(kg)
  • If weight is not entered, mL/kg/min is not computed and a guidance message is shown.

Pressure methodology (single source unit)

  • The pressure tab converts from one selected input unit and normalizes internally to kPa before generating all outputs.
  • Clinical outputs: mmHg, kPa, cmH₂O.
  • Gas supply outputs: psi, kPa, bar.

Pressure conversion constants

  • 1 mmHg = 0.133322 kPa, 1 kPa = 7.50062 mmHg
  • 1 mmHg = 1.35951 cmH₂O, 1 cmH₂O = 0.735559 mmHg
  • 1 psi = 6.89476 kPa, 1 bar = 100 kPa

Display and input behavior

  • Blank/invalid input clears computed pressure outputs.
  • Zero and negative values are allowed for pressure conversions when clinically relevant (e.g., sub-atmospheric readings).
  • Unit labels preserve standard casing as displayed (e.g., mmHg, kPa, cmH₂O, mL/min).
