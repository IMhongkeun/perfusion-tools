# Perfusion Tools

**A browser-based calculator suite for CPB & ECMO Perfusionists**

[perfusiontools.com](https://perfusiontools.com) is a lightweight, privacy-focused collection of web calculators designed for perfusionists working in cardiopulmonary bypass (CPB) and ECMO. All calculations run locally in the browser — no patient data is ever sent to a server.

---

## Calculators

| Tool                          | Description |
|-------------------------------|-------------|
| **BSA Calculator**            | Supports Mosteller, DuBois, Haycock, and other formulas with obesity adjustment options |
| **Goal-Directed Perfusion (GDP / DO₂i) Calculator** | Evaluates pump flow against oxygen delivery index targets |
| **Heparin Management**        | Weight-based CPB heparin dosing with obesity safety adjustments |
| **Predicted Hematocrit**      | Estimates post-dilution hematocrit considering circuit prime, blood products, and EBV |
| **Lean Body Mass (LBM) Calculator** | Boer, Hume, and other LBM estimation formulas |
| **Priming Volume Calculator** | Calculates tubing prime volume based on diameter and length |
| **Pediatric Echo Z-Score Calculator** | BSA-based normal range prediction for pediatric cardiac structures (aortic annulus, mitral valve, tricuspid valve, MPA/LPA/RPA, etc.) using PHN inverse prediction |
| **Time Calculator**           | Tracking for CPB time, cross-clamp time, total procedure time, and more |
| **Unit Converter**            | Flow rate, pressure, and other common unit conversions |
| **Quick Reference**           | Intraoperative reference for ACP, RCP, HCA, MUF, and other key concepts |

---

## Features
- **Mobile-friendly** — Fully optimized for use on smartphones and tablets in the OR and ICU
- **Transparent formulas** — Every calculator displays the exact equations and clinical references
- **Fully offline capable** — Static site that works without internet connection
- **Privacy by design** — No tracking, no data collection, no accounts, no server communication

---

## Who Is This For?
- Adult and pediatric cardiac surgery perfusionists
- ECMO Specialists
- Cardiac anesthesiologists
- Pediatric cardiologists (especially the Z-score tool)
